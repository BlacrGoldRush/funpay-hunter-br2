# FunPay Hunter для Black Russia

Бот для отслеживания новых предложений Black Russia на FunPay.

## Особенности
- ✅ Отслеживает только онлайн продавцов
- ✅ Фильтрует по ключевым словам "Black Russia"
- ✅ Проверяет цену (10-50000 руб)
- ✅ Отправляет уведомления в Telegram
- ✅ Работает 24/7 на Render

## Установка

1. Создайте нового бота через @BotFather в Telegram
2. Получите токен бота
3. Узнайте свой Chat ID (спросите у @userinfobot)
4. Создайте новый репозиторий на GitHub
5. Загрузите эти файлы
6. Создайте новый Web Service на Render:
   - Подключите GitHub репозиторий
   - Добавьте переменные окружения:
     - `TELEGRAM_BOT_TOKEN` = токен вашего бота
     - `TELEGRAM_CHAT_ID` = ваш chat ID
7. Запустите деплой

## Использование

1. Откройте ваш сервис на Render
2. Нажмите "Test Parsing" для проверки
3. Нажмите "Start Monitor" для запуска
4. В Telegram отправьте `/start` боту
5. Отправьте `/monitor` для запуска мониторинга

## Команды Telegram
- `/start` - начать работу
- `/check` - проверить сейчас
- `/monitor` - запустить мониторинг
- `/stop` - остановить мониторинг
- `/status` - статус
- `/help` - помощь
